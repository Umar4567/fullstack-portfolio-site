# Fullstack Portfolio Site

This project is a personal portfolio website built using React, showcasing the skills, experience, and projects of Umar Ahmed K M, a React and Full Stack Developer. The portfolio is designed to be responsive and visually appealing, utilizing Tailwind CSS for styling and Framer Motion for animations.

## Project Structure

```
fullstack-portfolio-site
├── public
│   └── Umar_Ahmed_Resume.pdf  # Resume PDF file available for download
├── src
│   ├── main.jsx                # Entry point of the React application
│   ├── App.jsx                 # Main application component
│   ├── fullstack_portfolio_site.jsx  # Portfolio component showcasing skills and projects
│   ├── index.css               # Global CSS styles
│   ├── components
│   │   └── Header.jsx          # Header component for navigation
│   └── assets                  # Directory for additional assets (images/icons)
├── package.json                # Project metadata and dependencies
├── vite.config.js              # Vite configuration file
├── postcss.config.cjs          # PostCSS configuration
├── tailwind.config.cjs         # Tailwind CSS configuration
├── .gitignore                  # Files and directories to ignore by Git
└── README.md                   # Project documentation
```

## Technologies Used

- **React**: For building the user interface.
- **Tailwind CSS**: For styling the application.
- **Framer Motion**: For animations and transitions.
- **Lucide React**: For icons used in the portfolio.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd fullstack-portfolio-site
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the portfolio.

## Features

- Responsive design that works on various screen sizes.
- Downloadable resume in PDF format.
- Showcase of skills, professional experience, and projects.
- Interactive elements with animations.
